# REFERENCE — Final Exam (Topic 13)

## Format

- **Items:** 100, random draw from module item banks
- **Timer:** 120 minutes (2 hours)
- **Passing score:** 80.00 / 100
- **Attempts:** 2 (second attempt gated by 7-day cooling period + completion of a targeted review assignment)
- **Proctoring:** Safe Exam Browser required
- **Shuffle:** question order shuffled; answer order shuffled
- **Feedback policy:** "After close only — show overall score; hide item-level feedback"
- **Review:** not available mid-attempt

## Remediation / retake

A student who fails:
1. Waits 7 calendar days from submission before being eligible to retake.
2. Completes a targeted review assignment in Moodle (scoped to their weakest-category misses).
3. Has 1 remaining attempt (max 2 total). Failing the 2nd attempt requires instructor contact and may delay cohort certification.

## Safe Exam Browser setup

- Configure SEB per Moodle docs; upload SEB config file to the quiz settings.
- End-to-end SEB test must pass Phase 2 and Phase 3 verification (see `/05-project-mgmt`).
- Provide SEB installation instructions to students in Topic 13's instruction Label.

## Accessibility accommodations

Documented per-student accommodations (extended time, screen reader, etc.) are configured via Moodle quiz overrides, not by changing the base exam settings. Keep an accommodation log separate from the gradebook.
