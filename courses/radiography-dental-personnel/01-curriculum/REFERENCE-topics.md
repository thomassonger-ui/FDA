## Module quiz rule

Each module quiz (Topics 0–12, 13 total) is 10–15 items randomized from a 25–30 item bank, **2 attempts, ≥ 80% to unlock the next module**.

## Unlock gating

Every topic is restricted by the prior topic's quiz completion (Moodle → Restrict Access → Activity completion of prior quiz). This enforces sequential progress.

## Where this list is used

- Moodle course shell — one Section per topic
- Gradebook — 13 module quizzes summed to 30% of course grade
- Final-exam composition — drawn proportionally from module item banks
- Certificate — lists course hours
- Pacing calendar — modules mapped to 6 weeks
- Public landing page — curriculum grid
