# REFERENCE — FDOE Standard 17 Crosswalk

Maps every course module to one or more Florida Department of Education benchmarks under Dental Assisting CIP 0351060108, Standard 17 (Radiographic Procedures).

## Crosswalk

| Benchmark | Description (summary) | Covered by |
|---|---|---|
| 17.01 | Fundamentals of radiation | M1, M2 |
| 17.02 | X-ray tubehead anatomy and production | M1 |
| 17.03 | Radiation protection (ALARA, MPD, shielding) | M3 |
| 17.04 | Film processing | M7 |
| 17.05 | Film mounting and handling | M7 |
| 17.06 | Intraoral techniques (paralleling, bisecting, ADA/FDA criteria) | M5, M6, M9 |
| 17.07 | Extraoral / panoramic imaging | M10, M11, M9 |
| 17.08 | Image quality and evaluation | M8 |
| 17.09 | Film duplication and storage | M7 |
| 17.10 | Record keeping | M7 |
| 17.11 | Error recognition and correction | M8, M9, M11 |
| 17.12 | Infection control and safety | M4, M12 |

Plus Standard 03 (legal and ethical responsibilities) → M12.

## Why the crosswalk matters

- **FDOE audit:** Auditors will ask which course content meets each benchmark. This table is your evidence.
- **Curriculum changes:** Before removing or renaming content, consult this table — dropping a topic may leave a benchmark uncovered.
- **Instructor orientation:** Give this to any new instructor as part of course onboarding.
