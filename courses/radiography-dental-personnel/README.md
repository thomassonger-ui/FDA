# Radiography for Dental Personnel

Full course unit — Moodle build for Florida's state-required dental radiography credential.

**Short name:** `RAD-DENT-PERS` · **Version:** 1.0 (April 2026)
**Regulatory authority:** FAC 64B5-9.011 · §466.017(5), F.S. · Florida Board of Dentistry
**FDOE alignment:** Dental Assisting CIP 0351060108 — Standard 17 (Radiographic Procedures)
**Contractor:** WorldTeachPathways dba WorldTeachESL / Atticus™
**Client:** Florida Institute of Dental Assisting

---

## At a glance

| | |
|---|---|
| Delivery | Hybrid — self-paced online + 1 day in-person clinical evaluation |
| Duration | **~64 hours** (56 online + 8 clinical) |
| Topics | 16 (Topic 0 Welcome → Topic 15 Certificate) |
| Content modules | 13 (M0–M12), each ≥ 80% passing |
| Final exam | 100 items, 120-min timer, Safe Exam Browser, 80% pass |
| Clinical evaluation | 7 competencies scored 1–4; passing = ≥ 3/4 on every competency |
| Gradebook weighting | Module quizzes 30% / Final 30% / Clinical rubric 40% |
| Access window | 90 days from cohort start |
| Cohort model | Manual + cohort-based; self-enrollment disabled |
| Retention | Minimum 4 years; nightly off-site backup |
