# Moodle

Reference docs and templates for the FIDA Moodle LMS build.

## Files

- [`REFERENCE-plugins.md`](./REFERENCE-plugins.md) — Required Moodle plugins with purpose and install notes.
- [`REFERENCE-cohorts.md`](./REFERENCE-cohorts.md) — Cohort/enrollment structure across all FIDA programs.
- [`REFERENCE-completion-rules.md`](./REFERENCE-completion-rules.md) — Activity and course completion rules, passing thresholds, certificate triggers.
- [`TEMPLATE-course-shell.md`](./TEMPLATE-course-shell.md) — Starter structure to copy when building a new Moodle course.

## Base Version & Theme

- **Moodle:** 4.x (LTS recommended)
- **Base theme:** Boost, customized with FIDA brand tokens (see `/marketing/REFERENCE-brand-spec.md`)
- **Hosting:** TBD — confirm with client before provisioning
