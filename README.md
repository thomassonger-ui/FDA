## File Naming Convention

- `README.md` — overview of the folder and how to use it
- `REFERENCE-*.md` — source-of-truth facts (regs, standards, canonical lists). Treat as read-only reference.
- `TEMPLATE-*.md` / `.html` — starter templates to copy when creating new deliverables
