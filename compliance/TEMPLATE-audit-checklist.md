# TEMPLATE — Audit Checklist

Run this before any FDOE, CIE, or Florida Board of Dentistry program review.

## Program documentation

- [ ] Current course catalog on file and published
- [ ] Enrollment agreement template matches what students sign
- [ ] Syllabus for each course reflects delivered content
- [ ] Instructor CVs and credentials on file
- [ ] Student complaint policy published and accessible

## Curriculum

- [ ] Every published learning objective is assessed
- [ ] Assessment items map back to stated objectives
- [ ] Content reviewed annually (date of last review recorded)
- [ ] For radiography: every FAC 64B5-9.011 topic represented

## Records

- [ ] Student transcripts complete for all enrolled students
- [ ] Certificate records (including unique IDs) retained for the required retention period
- [ ] Attendance/completion records exportable from Moodle
- [ ] Financial records (tuition paid, refunds) reconciled

## Technology / LMS

- [ ] Moodle backups running and verified recoverable
- [ ] User access logs retained per policy
- [ ] Certificate generation tested with a real student record

## Marketing / public statements

- [ ] Claims on website match what the program delivers (program length, content, outcomes)
- [ ] Accreditation/licensure claims are current and accurate
- [ ] Statistics cited (e.g., salary, placement) are sourced and dated
