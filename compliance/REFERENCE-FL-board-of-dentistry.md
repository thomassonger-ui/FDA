# REFERENCE — Florida Board of Dentistry

## Scope

The Florida Board of Dentistry (under the Department of Health) regulates the practice of dentistry and the roles of dental personnel in Florida. Relevant to FIDA because several FIDA offerings touch directly on Board-regulated credentials.

## Authoritative sources

- Florida Board of Dentistry: https://floridasdentistry.gov/
- Florida Administrative Code Chapter 64B5: https://www.flrules.org/

## Directly relevant rules

| Rule | Topic | FIDA program |
|---|---|---|
| FAC 64B5-9.011 | Dental radiography training | Radiography for Dental Personnel course |
| FAC 64B5-16 | Dental assistants — general | Dental Assisting entry-level + Expanded Functions |
| FAC 64B5-16.006 | Expanded functions of dental assistants | EFDA course |

(Confirm each rule reference is current at the Florida Administrative Code register before launch.)

## Course implications

- **Radiography:** certificate must reference FAC 64B5-9.011, issued by an acceptable training provider
- **Expanded Functions:** course must cover the functions defined in FAC 64B5-16.006 and require documented competency
- **Record retention:** Board may audit certification records; keep student records for at least the minimum retention period (confirm specific number with Board before launch)

## Communication / inquiries

- Any ambiguity about course-rule alignment should be confirmed with the Board of Dentistry prior to launch.
- Keep a log of any Board correspondence — this is your evidence trail if audited.
