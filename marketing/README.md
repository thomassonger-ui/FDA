# Marketing

Brand identity, outreach templates, and landing-page patterns for FIDA and its CE courses.

## Files

- [`REFERENCE-brand-audit.md`](./REFERENCE-brand-audit.md) — Audit of fldentalassisting.com: colors, fonts, logos, voice, positioning, conversion elements.
- [`REFERENCE-brand-spec.md`](./REFERENCE-brand-spec.md) — Normalized brand tokens (CSS variables, typography, button system, Moodle-region mapping).
- [`TEMPLATE-course-landing.html`](./TEMPLATE-course-landing.html) — On-brand single-file HTML landing page template for any CE course.
- [`TEMPLATE-email-outreach.md`](./TEMPLATE-email-outreach.md) — Cold/warm email templates for recruiting students and dental-office partners.

## When to use what

- **Starting a new course?** Copy `TEMPLATE-course-landing.html`, swap the course name, curriculum, and CTA.
- **Designing a Moodle theme?** Pull CSS tokens straight from `REFERENCE-brand-spec.md`.
- **Writing new copy?** Validate voice/tone against the "Voice" section of `REFERENCE-brand-audit.md`.

## Brand at a glance

- Primary: `#012D50` (navy) · CTA: teal gradient `#9EFCFF → #40CDD0`
- Headlines: Oswald · Body: Inter · Sub-nav: Roboto
- Voice: second-person, career-change friendly, aspirational-practical
